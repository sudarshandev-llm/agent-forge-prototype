## Installation Instructions

Add topics via the GitHub UI:

1. Go to your repository on GitHub.
2. Click the gear icon next to **About** on the right sidebar.
3. Enter topics (comma-separated) in the **Topics** field.
4. Click **Save**.

Alternatively, use the GitHub CLI:

```bash
gh repo edit your-org/agentforge --add-topic "ai-agents" --add-topic "agent-framework" # ... repeat for all 25
```

## Tips

- Topics are case-insensitive on GitHub.
- The first 20 topics carry the most SEO weight.
- Keep the description under 350 characters (GitHub truncates beyond that).
- Update topics as the project evolves and gains new capabilities.
- Add `hacktoberfest` annually during October to attract contributors.
